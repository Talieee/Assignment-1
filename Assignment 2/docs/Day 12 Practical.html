Question 1

Use a foreach loop to repeat 100 times:

generate a random sample from an exponential distribution with mean 1

calculate mean and variance

row-bind your results (rbind) (results = mean and variance).

Answer

#creates a column vector of all the means and variances for each 100 values sampled from an exponential distribution with mean 1 for each iteration
result =  foreach(i = 1:100, .combine = "rbind") %do%{
   exp_vec = rexp(100, 1)
   mean = mean(exp_vec)
   var = var(exp_vec)
   c(round(mean,2),round(var,2)) 
}
# changes the column and row names
colnames(result) <- c("Mean", "Variance")
rownames(result) <- c(seq(1,100))

#Put output into a table using kable
kable(result, caption = "Table 1 : Means and Variances for 100 exponential distributions with mean 1") %>% 
  kable_styling(bootstrap_options = "striped", "hover") %>% 
  scroll_box(., height = "300px")

set.seed(123)
c2 <- makeCluster(5) # assigned 5 workers 
registerDoParallel(c2)

coverage <- foreach(i = 1:1000, .combine = "rbind") %dopar%{
  values = rexp(50,1)

}

Question 4

The package iterators provides several functions that can be used to create sequences for the foreach function. For example, the irnorm function creates an object that iterates over vectors of normally distributed random numbers. It is useful when you need to use random variables drawn from one distribution in an expression that is run in parallel.

In this exercise, use the foreach and irnorm functions to iterate over 3 vectors, each containing 5 random variables. Find the largest value in each vector, and print those largest values.

Before running the foreach function set the seed to 1234.

Answer

set.seed(1234)
# creates 3 vectors with 5 vales from a normal distribution and then 
Large <- foreach(i = 1:3, .combine = 'rbind') %do% {
  vec = irnorm(n =5, count = 3)
  value = foreach(x = vec, .combine = c) %do% {
    x                                           # extracts each value x in each iteration
  }
  largest = max(value)
  c(round(largest, 2))             # Extracts the largest value from each iteration 
}

#changed row and column names
colnames(Large) <- c("Largest Value")
rownames(Large) <- c("Vector 1", "Vector 2", "Vector 3")

#produces a table with the output
Large %>% 
  kable(caption = "Table 2: Largest values of 3 vectors with normally distributed values") %>% 
  kable_styling(bootstrap_options = "striped", "hover", full_width = T)
